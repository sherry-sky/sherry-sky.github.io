# 第 1 章 分析化学导论

## 1.1 什么是分析化学

**分析化学（Analytical Chemistry）**是研究物质的组成、含量、结构和形态等化学信息的分析方法及相关理论的科学。

> 💡 **提示**：分析化学是化学学科的重要分支，被称为"科学的眼睛"，因为它为我们提供了认识物质世界的工具和方法。

### 分析化学的任务

1. **定性分析（Qualitative Analysis）**
   - 鉴定物质由哪些元素、原子团或化合物组成
   - 确定物质的化学结构

2. **定量分析（Quantitative Analysis）**
   - 测定物质中各组分的相对含量
   - 确定物质的纯度

3. **结构分析（Structural Analysis）**
   - 研究物质的分子结构
   - 确定原子的空间排列

## 1.2 分析化学的发展历程

| 时期 | 特点 | 代表方法 |
|------|------|----------|
| 古代（~1800）| 经验分析 | 感官判断、简单试验 |
| 近代（1800-1940）| 经典分析化学 | 滴定分析、重量分析 |
| 现代（1940-1980）| 仪器分析兴起 | 光谱、色谱、电化学 |
| 当代（1980-至今）| 分析科学 | 联用技术、微区分析、生物分析 |

## 1.3 分析方法的分类

### 按分析对象分类

- **无机分析**：分析无机物质（金属、矿物、盐类等）
- **有机分析**：分析有机物质（烃类、聚合物、生物分子等）

### 按测定原理分类

- **化学分析**：基于化学反应的分析方法
  - 滴定分析（酸碱、氧化还原、配位、沉淀）
  - 重量分析

- **仪器分析**：基于物理或物理化学性质的分析方法
  - 光学分析（吸收、发射、荧光）
  - 电化学分析（电位、伏安、电导）
  - 色谱分析（气相、液相、离子）
  - 质谱分析

### 按试样用量分类

| 方法 | 试样质量 (mg) | 试液体积 (mL) |
|------|---------------|---------------|
| 常量分析 | >100 | >10 |
| 半微量分析 | 10-100 | 1-10 |
| 微量分析 | 0.1-10 | 0.01-1 |
| 超微量分析 | <0.1 | <0.01 |

### 按组分含量分类

- **常量组分分析**：>1%
- **微量组分分析**：0.01% - 1%
- **痕量组分分析**：<0.01%

## 1.5 分析过程

一个完整的分析过程通常包括以下步骤：

```
1. 取样 → 2. 样品预处理 → 3. 测定 → 4. 数据处理 → 5. 结果报告
```

### 1. 取样（Sampling）

- 从总体中抽取有代表性的部分
- 取样方法直接影响分析结果的可靠性

### 2. 样品预处理（Sample Preparation）

- 溶解、消解
- 分离富集
- 衍生化

### 3. 测定（Measurement）

- 选择合适的分析方法
- 进行实验测量
- 记录原始数据

### 4. 数据处理（Data Processing）

- 误差分析
- 统计分析
- 结果计算

### 5. 结果报告（Reporting）

- 报告分析结果
- 说明不确定度
- 给出结论

## 本章小结

1. 分析化学是研究物质组成、含量、结构和形态的科学
2. 分析方法可按对象、原理、用量、含量等多种方式分类
3. 分析化学广泛应用于工业、科研、医疗、环境等领域
4. 完整的分析过程包括取样、预处理、测定、数据处理和报告
