# 第 10 章 Gran 作图法

## 10.1 Gran 作图法原理

### 概述

**Gran 作图法**（Gran Plot）是一种将滴定数据转换为线性关系的数学处理方法，由 G. Gran 于 1950 年提出。

### 基本思想

将滴定曲线的非线性部分转换为线性关系，通过外推法确定滴定终点。

### 优点

1. 不需要滴定到终点之后
2. 可以检测终点前的系统误差
3. 可以同时确定终点和平衡常数
4. 对数据噪声不敏感

## 10.2 强酸强碱滴定的 Gran 作图

### 强碱滴定强酸

滴定反应：$\text{H}^+ + \text{OH}^- \rightarrow \text{H}_2\text{O}$

**终点前**（$V < V_e$）：

溶液中剩余 H⁺，根据物料平衡：

$$[\text{H}^+] = \frac{c_0 V_0 - cV}{V_0 + V}$$

整理得 Gran 函数：

$$(V_0 + V) \cdot 10^{-\text{pH}} = K_a(V_e - V)$$

令 $F_1 = (V_0 + V) \cdot 10^{-\text{pH}}$，则：

$$F_1 = K_a(V_e - V)$$

以 $F_1$ 对 $V$ 作图，得直线，外推至 $F_1 = 0$，得 $V = V_e$。

**终点后**（$V > V_e$）：

溶液中过量 OH⁻：

$$[\text{OH}^-] = \frac{c(V - V_e)}{V_0 + V}$$

令 $F_2 = (V_0 + V) \cdot 10^{\text{pH}}$，则：

$$F_2 = \frac{K_w}{c}(V - V_e)$$

以 $F_2$ 对 $V$ 作图，得直线，外推至 $F_2 = 0$，得 $V = V_e$。

### 作图步骤

1. 记录滴定过程中各点的 V 和 pH
2. 计算 Gran 函数 $F_1$ 或 $F_2$
3. 以 F 对 V 作图
4. 线性拟合，外推求 $V_e$

## 10.3 弱酸滴定的 Gran 作图

### 强碱滴定弱酸

滴定反应：$\text{HA} + \text{OH}^- \rightarrow \text{A}^- + \text{H}_2\text{O}$

**终点前**：

根据 Henderson-Hasselbalch 方程：

$$\text{pH} = \text{p}K_a + \lg\frac{[\text{A}^-]}{[\text{HA}]}$$

$$[\text{H}^+] = K_a \frac{[\text{HA}]}{[\text{A}^-]} = K_a \frac{c_0 V_0 - cV}{cV}$$

整理得：

$$(V_0 + V)[\text{H}^+] = K_a \frac{V_0 + V}{V}(V_e - V)$$

当 $V$ 接近 $V_e$ 时，可近似为：

$$V \cdot 10^{-\text{pH}} \propto (V_e - V)$$

### 同时求 $K_a$ 和 $V_e$

从 Gran 图的斜率和截距可以同时求得：
- 终点体积 $V_e$
- 酸解离常数 $K_a$

## 10.4 沉淀滴定的 Gran 作图

### 银量法测定 Cl⁻

滴定反应：$\text{Ag}^+ + \text{Cl}^- \rightarrow \text{AgCl}\downarrow$

**终点前**：

$$[\text{Cl}^-] = \frac{c_0 V_0 - cV}{V_0 + V}$$

用 Cl⁻ 选择性电极测定 pCl：

$$(V_0 + V) \cdot 10^{-\text{pCl}} \propto (V_e - V)$$

**终点后**：

$$[\text{Ag}^+] = \frac{c(V - V_e)}{V_0 + V}$$

用 Ag 电极或 Cl⁻ 电极：

$$(V_0 + V) \cdot 10^{\text{pAg}} \propto (V - V_e)$$

## 10.5 多元酸碱的 Gran 作图

### 二元弱酸的滴定

对于 H₂A，有两个终点 $V_{e1}$ 和 $V_{e2}$。

**第一终点前**：

处理类似于弱酸滴定，可求 $K_{a1}$ 和 $V_{e1}$。

**第一和第二终点之间**：

$$[\text{H}^+] = K_{a2} \frac{[\text{HA}^-]}{[\text{A}^{2-}]}$$

可求 $K_{a2}$ 和 $V_{e2}$。

### 混合酸的滴定

对于 HA + HB 混合酸，若 $K_{a1}/K_{a2}$ 足够大，可分别滴定。

Gran 作图可分别确定两个终点。

## 10.6 Gran 作图法的应用

### 确定滴定终点

 Gran 作图法特别适用于：
1. 终点不明显的滴定
2. 弱酸弱碱滴定
3. 稀溶液滴定
4. 有色或浑浊溶液

### 测定平衡常数

从 Gran 图的斜率可求得：
- 酸解离常数 $K_a$
- 溶度积 $K_{sp}$
- 稳定常数 $K_f$

### 检测系统误差

如果 Gran 图不是直线，说明存在：
- 副反应
- 离子强度变化
- 电极响应问题

## 10.7 使用 Excel 进行 Gran 作图

### 数据处理步骤

1. **输入原始数据**
   - A 列：滴定体积 V（mL）
   - B 列：pH 值

2. **计算 Gran 函数**
   - C 列：$F_1 = (V_0 + V) \times 10^{-\text{pH}}$
   - 公式：`=(V0+A2)*10^(-B2)`

3. **绘制散点图**
   - X 轴：V
   - Y 轴：F₁

4. **添加趋势线**
   - 选择线性拟合
   - 显示公式和 R²

5. **计算终点**
   - 令 Y = 0，从公式求 X 截距
   - $V_e = -\text{截距}/\text{斜率}$

### 示例公式

```
终点前 Gran 函数：=(V0+A2)*10^(-B2)
终点后 Gran 函数：=(V0+A2)*10^(B2-14)
```

## 10.8 Gran 作图法的注意事项

### 数据选择

1. 选择线性好的数据段
2. 避开缓冲区域
3. 终点附近数据更重要

### 离子强度影响

1. 高离子强度下活度系数变化
2. 可加入惰性电解质保持离子强度恒定

### 温度影响

1. 温度影响 $K_w$ 和 $K_a$
2. 实验过程保持温度恒定

### 电极响应

1. 确保电极响应正常
2. pH 电极斜率应在 56-60 mV/pH

## 本章小结

1. Gran 作图法是将滴定数据线性化的数学方法
2. 通过外推法可准确确定滴定终点
3. 可同时测定平衡常数
4. 适用于终点不明显的滴定
5. Excel 可方便地进行 Gran 作图
