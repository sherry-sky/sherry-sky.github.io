# 第 5 章 线性回归分析

## 5.1 相关与回归

### 函数关系与相关关系

- **函数关系**：变量之间有确定的对应关系
  - 例如：$c = \frac{n}{V}$

- **相关关系**：变量之间有关系，但不是确定的
  - 例如：身高与体重
  - 分析化学中：浓度与吸光度

### 相关的分类

| 类型 | 说明 | 图示 |
|------|------|------|
| 正相关 | 一变量增加，另一变量也增加 | ↗ |
| 负相关 | 一变量增加，另一变量减少 | ↘ |
| 不相关 | 变量之间无关系 | · · · |

## 5.2 一元线性回归方程

### 回归方程的形式

$$y = a + bx$$

其中：
- $y$：因变量（响应变量）
- $x$：自变量（解释变量）
- $a$：截距
- $b$：斜率

### 最小二乘法原理

最小二乘法是使残差平方和最小的方法。

**残差**：测定值与回归值的差

$$Q = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 = \sum_{i=1}^{n} (y_i - a - bx_i)^2$$

令 $\frac{\partial Q}{\partial a} = 0$ 和 $\frac{\partial Q}{\partial b} = 0$，可求得 $a$ 和 $b$。

### 回归系数的计算

**斜率**：

$$b = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2} = \frac{\sum x_i y_i - n\bar{x}\bar{y}}{\sum x_i^2 - n\bar{x}^2}$$

**截距**：

$$a = \bar{y} - b\bar{x}$$

> 💡 **提示**：回归直线一定经过点 $(\bar{x}, \bar{y})$

## 5.3 相关系数

### 相关系数的定义

**相关系数** $r$ 用于衡量两变量之间线性相关的程度：

$$r = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n} (x_i - \bar{x})^2 \cdot \sum_{i=1}^{n} (y_i - \bar{y})^2}}$$

### 相关系数的性质

1. **取值范围**：$-1 \leq r \leq 1$

2. **意义**：
   | $r$ 值 | 相关程度 |
   |--------|----------|
   | $r = 1$ | 完全正相关 |
   | $r = -1$ | 完全负相关 |
   | $r = 0$ | 不相关 |
   | $|r| > 0.9$ | 高度相关 |
   | $0.7 < |r| < 0.9$ | 显著相关 |
   | $|r| < 0.5$ | 低度相关 |

3. **符号**：$r > 0$ 为正相关，$r < 0$ 为负相关

### 相关系数的显著性检验

查相关系数临界值表，若 $|r| > r_{\text{临界}}$，则相关显著。

## 5.4 校准曲线

### 校准曲线的制作

在仪器分析中，常用校准曲线（工作曲线）进行定量分析。

**步骤**：
1. 配制一系列标准溶液
2. 测定各标准溶液的响应值
3. 以浓度为横坐标，响应值为纵坐标作图
4. 用最小二乘法拟合回归方程
5. 测定未知样品的响应值，代入方程求浓度

### 校准曲线的线性范围

- 在校准曲线的线性范围内使用
- 超出线性范围会导致结果不准确

### 截距的意义

- **理想情况**：截距 $a = 0$
- **实际情况**：截距可能不为 0
  - $a > 0$：可能存在空白值
  - $a < 0$：可能存在基体效应

> 💡 **提示**：制作校准曲线时，应包含空白溶液。

## 5.5 回归分析的应用

### 未知样品的测定

由回归方程 $y = a + bx$，得：

$$x_{\text{未知}} = \frac{y_{\text{未知}} - a}{b}$$

### 灵敏度和检出限

**灵敏度**：校准曲线的斜率 $b$

**检出限**（Limit of Detection, LOD）：

$$\text{LOD} = \frac{3s_b}{b}$$

其中 $s_b$ 为空白测定的标准偏差。

**定量限**（Limit of Quantitation, LOQ）：

$$\text{LOQ} = \frac{10s_b}{b}$$

### 回归方程的精度

**剩余标准偏差**：

$$s_{y/x} = \sqrt{\frac{\sum (y_i - \hat{y}_i)^2}{n-2}}$$

**预测值的标准偏差**：

$$s_{\hat{y}} = s_{y/x} \sqrt{\frac{1}{n} + \frac{(x_0 - \bar{x})^2}{\sum (x_i - \bar{x})^2}}$$

> 💡 **注意**：$x_0$ 越接近 $\bar{x}$，预测精度越高。

## 5.6 使用 Excel 进行回归分析

### 方法一：添加趋势线

1. 选中数据，插入散点图
2. 右键点击数据点，选择"添加趋势线"
3. 选择"线性"，勾选"显示公式"和"显示 R 平方值"

### 方法二：LINEST 函数

```
=LINEST(known_y's, known_x's, TRUE, TRUE)
```

返回数组包含：斜率、截距、标准偏差、R²等。

### 方法三：数据分析工具

1. 数据 → 数据分析 → 回归
2. 输入 Y 范围和 X 范围
3. 输出回归统计结果

## 本章小结

1. 最小二乘法是拟合回归直线的标准方法
2. 相关系数 r 衡量线性相关程度
3. 校准曲线是仪器分析中常用的定量方法
4. 回归方程可用于预测和未知样品测定
5. 灵敏度和检出限是分析方法的重要指标
