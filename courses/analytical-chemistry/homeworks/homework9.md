- **原电池**：将化学能转化为电能的装置

  - 阴极 (Cathode)：发生还原反应，电子进入

- **电子流向**：从负极（阳极）流向正极（阴极）
- **电流流向**：从高电位流向低电位

## ✅ 参考答案

<!-- ANSWER_START for="H9-1" -->
**H9-1.** 原电池分析

**阳极 (Anode)：** 在原电池中，阳极是发生氧化反应的电极，电子从阳极离开。

**阴极 (Cathode)：** 发生还原反应的电极，电子进入阴极。

通常，阳极放在左侧，阴极放在右侧。

**电子流向：** 电子通过电路从负极流向正极。

**电流流向：** 电流总是从高电位流向低电位。

**电池电位：** 电池电位等于阴极的还原电位 (Ag⁺|Ag) 减去阳极的还原电位 (Cd²⁺|Cd)（注意：不是 Cd|Cd²⁺）。
<!-- ANSWER_END -->

<!-- ANSWER_START for="H9-2" -->
**H9-2.** 参比电极特性

**答案：abce**

**分析：**
- a. 正确 - 参比电极应具有普遍接受性
- b. 正确 - 应易于构建
- c. 正确 - 应符合能斯特方程
- d. 错误 - 参比电极应能在特定条件下提供恒定电位（如饱和溶液使 log 项为常数）
- e. 正确 - 应能在特定条件下提供恒定电位

**答案：** **abce**

**H9-3.** 铅酸电池反应

**a. 阳极反应：**
$$\text{Pb(s)} + \text{SO}_4^{2-} \rightleftharpoons \text{PbSO}_4\text{(s)} + 2\text{e}^-$$

**b. 阴极反应：**

**c. 总反应：**
$$\text{Pb(s)} + \text{PbO}_2\text{(s)} + 2\text{H}_2\text{SO}_4 \rightleftharpoons 2\text{PbSO}_4\text{(s)} + 2\text{H}_2\text{O}$$

**H9-4.** 电池电位计算

**a. 电极反应：**

阳极：$\text{Pb} + 2\text{F}^- \rightleftharpoons \text{PbF}_2\text{(s)} + 2\text{e}^-$

阴极：$\text{AgCl} + \text{e}^- \rightleftharpoons \text{Ag} + \text{Cl}^-$

总反应：$\text{Pb} + 2\text{F}^- + 2\text{AgCl} \rightleftharpoons \text{PbF}_2\text{(s)} + 2\text{Ag} + 2\text{Cl}^-$

**b. 计算电池电压：**

$$E_{\text{cell}} = E^\circ(\text{Pb}|\text{PbF}_2) + E^\circ(\text{AgCl}|\text{Ag}) - \frac{0.05916}{2} \log\frac{[\text{Cl}^-]^2}{[\text{F}^-]^2}$$

$$= -E^\circ(\text{PbF}_2|\text{Pb}) + E^\circ(\text{AgCl}|\text{Ag}) - \frac{0.05916}{2} \log\frac{(0.1\ \text{M})^2}{(0.1\ \text{M})^2}$$

$$= -(-0.350\ \text{V}) + 0.222\ \text{V} - 0 = 0.572\ \text{V}$$

**c. 电子流向：** 电子从 Pb 电极流向 Ag 电极。

**答案：**
- **电池电压：0.572 V**
- **电子从 Pb 流向 Ag**
<!-- ANSWER_END -->

<!-- ANSWER_START for="H9-5" -->
**H9-5.** 原电池计算

**a. 计算半电池电位和电池电位：**

**左电极：**
$$\text{Br}_2\text{(l)} + 2\text{e}^- \rightleftharpoons 2\text{Br}^-$$

$$E_{\text{left}} = E^\circ - \frac{0.05916}{2} \log[\text{Br}^-]^2 = 1.078\ \text{V} - \frac{0.05916}{2} \log(0.10\ \text{M})^2 = 1.137\ \text{V}$$

**右电极：**
$$\text{Al}^{3+} + 3\text{e}^- \rightleftharpoons \text{Al}$$

$$E_{\text{right}} = E^\circ - \frac{0.05916}{3} \log\frac{1}{[\text{Al}^{3+}]} = -1.677\ \text{V} - \frac{0.05916}{3} \log\frac{1}{0.01\ \text{M}} = -1.716\ \text{V}$$

**电池电位：**
$$E_{\text{cell}} = E_{\text{right}} - E_{\text{left}} = -1.716\ \text{V} - 1.137\ \text{V} = -2.853\ \text{V}$$

**b. 电子流向：** 电子从 Al 电极流向 Pt 电极（从右到左）。电池电位为负值说明这个表示法与默认形式相反。

**c. 自发电池总反应：**
$$\text{Al} + \frac{3}{2}\text{Br}_2\text{(l)} \rightleftharpoons \text{Al}^{3+} + 3\text{Br}^-$$

**d. 限制试剂判断：**

Br₂(l) 的物质的量：
$$n(\text{Br}_2) = \frac{14.3\ \text{mL} \times 3.12\ \text{g/mL}}{79.904\ \text{g/mol}} = 0.2792\ \text{mol}$$

Al 的物质的量：
$$n(\text{Al}) = \frac{12.0\ \text{g}}{26.981\ \text{g/mol}} = 0.4448\ \text{mol}$$

1 mol Al 需要 1.5 mol Br₂(l)：
$$0.4448\ \text{mol Al 需要} 0.6672\ \text{mol Br}_2 > 0.2792\ \text{mol}$$

**答案：** **Br₂(l) 是限制试剂**
